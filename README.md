## Description

This is my personal portfolio website built with Next.js, TypeScript, and Tailwind CSS. It showcases my skills, projects, and experience as a Full Stack Software Developer.

## Technologies Used

- **Framework:** Next.js 14
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Icons:** Heroicons

## Getting Started

1. Clone the repository
2. Install dependencies: `yarn install`
3. Run the development server: `yarn dev`
4. Open [http://localhost:3000](http://localhost:3000) in your browser

## Deployment

The site is deployed on Vercel. To deploy your own version, connect your GitHub repository to Vercel and it will automatically build and deploy.
